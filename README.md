# Café Lindenhof (Demo)

Café Lindenhof ist eine fiktive Demo-Website für ein kleines Nachbarschafts-Café.  
Die Seite legt den Schwerpunkt auf Atmosphäre, Öffnungszeiten, Lage und einfache Kontaktmöglichkeiten.

- Live: https://heimhafen.github.io/cafe-lindenhof/
- Tech-Stack: HTML, CSS, leichtes JavaScript – statische Auslieferung über GitHub Pages.
- Inhalte: Hero mit Nutzenversprechen, Angebot (Kaffee, Kuchen, Snacks), Atmosphäre, Öffnungszeiten & Lage, Kontakt/Reservierung.
- Zweck: Zweites Showcase im Portfolio von HeimHafen Web, um Gestaltung und Struktur für lokale Betriebe (Cafés, kleine Läden etc.) zu zeigen.
