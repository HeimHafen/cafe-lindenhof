# Café Lindenhof (Demo)

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Online-b0553e)](https://heimhafen.github.io/cafe-lindenhof/)
![Stack](https://img.shields.io/badge/Stack-HTML%20%7C%20CSS%20%7C%20JS-lightgrey)
![Status](https://img.shields.io/badge/Status-Demo%20Showcase-brightgreen)

Café Lindenhof ist eine fiktive Demo-Website für ein kleines Nachbarschafts-Café.  
Die Seite legt den Schwerpunkt auf Atmosphäre, Öffnungszeiten, Lage und einfache Kontaktmöglichkeiten.

- **Live:** https://heimhafen.github.io/cafe-lindenhof/
- **Tech-Stack:** HTML, CSS, leichtes JavaScript – statisch über GitHub Pages veröffentlicht.
- **Inhalte:** Hero mit Nutzenversprechen, Angebot (Kaffee, Kuchen, Snacks), Atmosphäre, Öffnungszeiten & Lage, Kontakt/Reservierung.
- **Zweck:** Zweites Showcase im Portfolio von HeimHafen Web, um Gestaltung und Struktur für lokale Betriebe (Cafés, kleine Läden etc.) zu zeigen.

## Seitenstruktur

- Hero-Bereich mit Fokus auf ruhige Atmosphäre und Nachbarschaft
- Abschnitt „Angebot“ (Kaffeespezialitäten, Kuchen, kleine Speisen)
- Bereich „Atmosphäre“ mit Text- und Listenbeschreibung statt großer Bildstrecken
- Öffnungszeiten, Adresse, Anreise (ÖPNV, Parken)
- Kontaktformular für Reservierungsanfragen (in der Demo nicht angebunden)

Das Design unterscheidet sich bewusst vom Camping-Showcase: wärmere Farbwelt, andere Stimmung, aber gleicher Qualitätsanspruch.
